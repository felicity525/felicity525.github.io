## C++ STL
### 什么是STL🧐
STL即标准模板库（Standard Template Library），是C++标准库的一部分，提供了很多可直接使用的数据结构和算法。一般算法竞赛都支持STL库的使用，合理使用STL可以避免大量的造轮子工作，提高效率。
### STL容器
#### 迭代器
迭代器用于访问STL容器中的元素（类似指针
`#include<iterator>`
**常用运算符**
- 自增++自减--迭代器前后移动1位
- +或-整型前后移动指定位数
- 解引用*用于获取或修改它指向的元素
- 头迭代器begin()可以获得容器第一个元素的迭代器
- 尾迭代器end()可以获得指向容器尾端的迭代器（“尾端”不是最后一个元素，可以视为最后一个元素的后继）
示例：使用迭代器遍历vector
```cpp
for(vector<int>::iterator it = a.begin();it!=a.end();it++)
{
    cout<<*it<<" ";
}
```
使用auto关键字简化迭代器类型
```cpp
    for (auto it = a.begin(); it != a.end(); it++)
    {
        cout << *it << " ";
    }
```
C++11之后支持
```cpp
for(int num : a)
{
    cout<<num<<" ";
}
```
#### vector
- 头文件`#include<vector>`
- 构造函数`vector<int> a(n,m)`创建一个元素类型为`int`，大小为`n`，元素初值为`m`的一维数组`a`（时间复杂度$$O(n)$$，和数组大小成线性关系)
- 访问元素（时间复杂度$$O(1)$$）
  - `v.at(pos)`返回下标为`pos`的引用
  - `v[pos]`返回下标为`pos`的引用
  - `v.front() v.back()`返回首元素、尾元素的引用
- 获取数组大小`a.size()`
- 重新分配数组大小`a.resize()`
- 末尾添加新元素`a.push_back(m)`
  - 平均时间复杂度为$$O(1)$$
  - 最坏时间复杂度$$O(N)$$（若超出当前分配内存的大小则重新分配内存并复制已有数据）
- 删除末尾元素`a.pop_back()`（时间复杂度为$$O(1)$$）
- 判断数组是否为空`a.empty()`返回`bool`类型
- 清空数组`a.clear()`
#### stack
栈（先进后出）
- 头文件`#include<stack>`
- 创建一个栈`stack<int> s`
- 访问栈顶元素`s.top()`（时间复杂度为$$O(1)$$）
- 进栈`s.push(data)`元素压入栈顶（时间复杂度为$$O(1)$$）
- 出栈`s.pop()`栈顶元素出栈（时间复杂度为$$O(1)$$）
- 获取大小`s.size()`
- 判空`s.empty()`
#### queue
队列（先进先出）
- 头文件`#include<queue>`
- 创建队列`queue<int>q`
- 访问队首`q.front()`（时间复杂度为$$O(1)$$）   
- 访问队尾`q.end()`（时间复杂度为$$O(1)$$）
- 入队`q.push(data)`元素压入队尾（时间复杂度为$$O(1)$$）
- 出队`q.pop()`队首元素出队（时间复杂度为$$O(1)$$）
- 获取大小`q.size()`
- 判空`q.empty()`
#### deque
双端队列（两端都可以出队入队）
- 头文件`#include<deque>`
- 创建`deque<int>de`（时间复杂度$$O(n)$$，和容器大小成线性关系)
- 访问：`de.front() de.back()`返回队首\队尾元素的引用 （时间复杂度为$$O(1)$$）
- 出队：队尾元素出队`de.pop_back()` 队首元素出队`de.pop_front()`（时间复杂度为$$O(1)$$）
- 入队：压入队首`de.push_front()` 压入队尾`de.push_back()`（时间复杂度为$$O(1)$$）
- 获取大小`de.size()`
- 判空`de.empty()`
#### priority_queue
优先队列（默认为大顶堆，即队列顶部为最大的元素）：按照元素的优先级进行排序，（默认）每次出队操作总是弹出优先级最高的元素
- 头文件`#include<priority_queue>`
- 创建`priority_queue<int>pq`默认为大顶堆
  - 提供比较函数实现自定义类型的优先队列
  ```cpp
    struct cmp
    {
        bool operator()(int a, int b)
        {
            return a > b;
        }
    };
    priority_queue<int, vector<int>, cmp> pq2;//小顶堆
    ```

```cpp
struct cmp2
    {
        bool operator()(pair<int, int> a, pair<int, int> b)
        {
            return a.second > b.second;
        }
    };
    priority_queue<pair<int, int>, vector<pair<int, 
```
- 插入`pq.push(val)`将元素`val`插入优先队列并重新调整堆以保证优先级顺序，时间复杂度近似为$$O(log n)$$
- `pq.pop()`移除队列顶部的元素（即移除优先级最高的元素），时间复杂度近似为$$O(log n)$$
- 访问队列顶部元素`pq.top()` 返回优先级最高的元素的引用，（时间复杂度为$$O(1)$$）
- 获取大小`pq.size()`（时间复杂度为$$O(1)$$）
- 判空`pq.empty()`（时间复杂度为$$O(1)$$）
#### set
* `set`、`multiset`和`map`通常采用红黑树实现，其插入、查看和删除操作的时间复杂度近似为$$O(log n)$$
类似数学中的集合，set中不会出现值相同的元素，默认从小到大排序
- 头文件`#include<set>`
- 创建`set<int>s`
- 插入`s.insert(x)`
- 删除（时间复杂度为$$O(log n)$$）
  - `erase(x)`删除值为x的所有元素，返回删除元素的个数
  - `erase(pos)`删除迭代器为pos的元素
  - `erase(first,last)`删除迭代器在$$[first,last)$$范围内的所有元素
- 获取大小`s.size()`
- 清空`s.clear()`
- 查找
  - `count(x)`返回set内键为x的元素数量
  - `find(x)`返回set内键为x的元素的迭代器，若不存在该元素则返回end()
  - `lower_bound(x)`返回指向首个不小于x的元素的迭代器，若不存在则返回end()
  - `upper_bound(x)`返回指向首个大于x的元素的迭代器，若不存在则返回end()
- 判空`s.empty()`
- 自定义比较方式
  ```cpp
    struct cmp
    {
        bool operator()(int a, int b)
        {
            return a > b;
        }
    set<int,  cmp> s;//从大到小排列
  ```

#### string
- 头文件`#include<string>`
- 构造`string s`创建一个空字符串s
- 初始化`string s(10,'0')`创建一个长度为10每个字符都为0的字符串
- 赋值：用=赋值`string s = "hello world!"`
- 读入`cin>>s`输出`cout<<s`
- 字符串拼接`s = s1 + s2`，时间复杂度$$O(n)$$（n为拼接后字符串的长度）
- 获取字符串长度`s.length()`，时间复杂度为$$O(1)$$
- 截取字符串`s1 = s.substr(n,m)`从下标为n的字符开始截取m个字符，若m缺省则截取下标为n及其后面的所有字符，时间复杂度为$$O(m)$$
- 插入字符串，时间复杂度$$O(n + m)$$
  - `s.insert(index,n,ch)`在`index`处连续插入n次字符串ch
  - `s.insert(index,str)`在`index`处插入字符串str
- 删除字符串`s.erase(index,n)`删除从下标为`index`开始（包含）的n个字符，若n缺省则删去index及其后面的所有字符，时间复杂度$$O(n)$$
- 查找`s.find(str,pos)`查找字符串s中在pos之后（包含）第一次出现str子串的位置（子串第一个元素的位置），若不存在则返回`string::npos`
*`s.find(str,pos)`时间复杂度$$O(nm)$$，n为字符串总长，m为查询子串的长度，相当于暴力查询，不建议用`find()`解决时间复杂度要求较高的问题
- 替换字符串`s.replace(n,m,str)`将从下标为n开始的m个字符的子串替换为str，时间复杂度$$O(n)$$

### STL算法
STL提供了很多实现算法的模板函数，基本都包含在`<algorithm>`头文件中。
- **find**： `find(v.begin(),v.end(),value)`顺序查找值为value的元素
- **swap**：交换两个变量的值`swap(a,b)`
- **reverse**：翻转数组、字符串,时间复杂度$$O(n)$$
  - `reverse(v.begin(),v.end())`反转整个容器
  - `reverse(v.begin() + 1, v.begin() + 3)`对容器下标$$[1,3)$$部分进行反转
- **sort**：排序，默认将元素从小到大排序，也可以通过自定义比较函数cmp实现复杂比较,时间复杂度$$O(n log n)$$
  - `sort(v.begin(),v.end(),cmp)`对整个容器排序
  - `sort(v.begin() + 1, v.begin() + 3, cmp)`对容器下标$$[1,3)$$部分进行排序
- **merge**
将两个有序区间有序合并到第三个序列的迭代器上
```cpp
vector<int>v1 = {1,3,5};
vector<int>v2 = {2,4,6};
vector<int>res(6);
merge(v1.begin(),v1.end(),v2.begin(),v2.end(),res.begin());
```


- **max和min**：`max(a,b)`
  求多个元素的最大值或最小值：`max(a,max(b,c))`对于c++11之后`max({a,b,c}`)
## 常见排序算法
### 冒泡排序
算法步骤（升序排序为例）
1.从第一个元素开始，比较相邻的两个元素，若前一个元素更大，则交换两个元素；
2.对每一对相邻元素重复上述步骤，直到容器末尾。此时最大的元素已被冒泡到容器末尾
3.忽略已排序的最后一个元素，重复上述步骤，直至排序完成

```cpp
vector<int> v = {5, 3, 4, 2, 1};
for (int i = 0; i < v.size(); i++)
{
    for (int j = 0; j < v.size() - i - 1; j++)
    {
        if (v[j] > v[j + 1])
            swap(v[j], v[j + 1]);
    }
}
```

时间复杂度$$O(n^2)$$ （两层for循环遍历数组）对于大规模数据效率较低
空间复杂度$$O(1)$$ (原地排序不需要额外的存储）
### 选择排序
算法步骤（升序排序为例）
1.将容器分为已排序部分和未排序部分（初始时已排序部分为空）
2.在未排序部分中找到最小的元素，与未排序部分的第一个元素交换位置
3.将未排序部分的起始位置向后移动一位，扩大已排序部分的范围
4.重复上述步骤直至未排序部分为空

```cpp
void selectionSort(vector<int>&arr, int n)
{
    int i, j, minn;
    for (i = 0; i < n - 1; i++)
    {
        minn = i;
        for (j = i + 1; j < n; j++)
            if (arr[j] < arr[minn])
                minn= j;
        swap(arr[minn], arr[i]);
    }
}
```
 
时间复杂度$$O(n^2)$$（两层for循环遍历数组）对于大规模数据效率较低
空间复杂度$$O(1)$$ (原地排序不需要额外的存储）

### 插入排序

算法步骤（升序排序为例）
1.将容器分为已排序部分和未排序部分（初始时已排序部分只包含第一个元素）
2.在未排序部分中取出第一个元素，将该元素与已排序部分从后往前依次比较，找到合适的位置插入
3.重复上述步骤直至未排序部分为空

```cpp
void insertionSort(vector<int>&arr, int n)
{
    int i, j, key;
    for (i = 1; i < n; i++)
    {
        key = arr[i];
        j = i - 1;
        while (j >= 0 && arr[j] > key)
        {
            arr[j + 1] = arr[j];
            j = j - 1;
        }
        arr[j + 1] = key;
    }
}
```
 
时间复杂度：
最坏$$O(n^2)$$原列表完全逆序，每次插入都需要移动所有已排序的元素
最好$$O(n)$$原列表有序，只需遍历一次列表
空间复杂度$$O(1)$$ (原地排序不需要额外的存储）

### 归并排序

算法步骤（升序排序为例）
1.将容器对半分为两个子数组
2.递归地对每个子数组进行排序
3.将两个已排序的子数组合并为一个有序数组
合并操作
1.申请空间存放待合并的数组
2.设定两个指针，最初分别指向两个待合并数组的起始位置
3.比较两个指针所指向的元素，选择较小的放入原数组，并移动指针到下一位置
4.重复步骤3直至某一指针到达数组末尾
5.将未到达末尾的数组的剩余元素复制到原数组

```cpp
void merge(vector<int> &arr, int start, int mid, int end)
{
    int l = 0, r = 0, k = start;
    vector<int> left(arr.begin() + start, arr.begin() + mid + 1);
    vector<int> right(arr.begin() + mid + 1, arr.begin() + end + 1);
    while (l < left.size() && r < right.size())
    {
        if (left[l] <= right[r])
        {
            arr[k] = left[l];
            l++;
        }
        else
        {
            arr[k] = right[r];
            r++;
        }
        k++;
    }
    //复制剩余元素
    while(l<left.size())
    {
        arr[k] = left[l];
        l++;
        k++;
    }
    while(r<right.size())
    {
        arr[k] = right[r];
        r++;
        k++;
    }
}
void mergeSort(vector<int> &arr, int start, int end)
{
    if (start >= end)
        return;
        int mid = (start + end) / 2;
        mergeSort(arr, start, mid);//递归地处理左右子数组
        mergeSort(arr, mid + 1, end);
        merge(arr, start, mid, end);
}
//调用mergeSort(arr, 0, n - 1);
```
 
时间复杂度$$O(n log n)$$
分解：每次对半分，需要执行$$O(log n)$$次
合并：$$O(n)$$
空间复杂度$$O(n)$$（合并操作需要额外空间来存储）

### 快速排序

算法步骤（升序排序为例）
1.选择一个元素作为基准
2.将所有小于基准元素的元素放在基准的左侧，所有大于基准元素的元素都放在基准的右侧
3.递归地对基准元素左侧和右侧的子数组进行快速排序

```cpp
void quicksort(vector<int> &arr, int left, int right)
{
    if (left >= right)
        return;
    int i = left, j = right + 1;
    int key = arr[left];//选取第一个元素作为基准
    while (i < j)
    {
        while (++i <= right && arr[i] < key)
            ;
        while (--j > left && arr[j] > key)
            ;
        if (i >= j)
            break;
        swap(arr[i], arr[j]);
    }
    swap(arr[left], arr[j]);//放置基准元素
    quicksort(arr, left, j - 1);//递归处理左右子数组
    quicksort(arr, j + 1, right);
}
```
 
时间复杂度$$O(n log n)$$
分解：每次对半分，需要执行$$O(log n)$$次
合并：$$O(n)$$
空间复杂度$$O(log n)$$（原地排序，递归调用函数需要额外$$O(log n)$$的空间）

### 希尔排序

对普通插入排序的优化
算法步骤（升序排序为例）
1.选择一个增量序列（常用`n/2,n/4,...1`)，用于将数组分成若干子数组
2.按照增量序列分组后，对每个子列表进行插入排序
3.逐步缩小增量，重复上述步骤直到增量为1
4.当增量为1时，对整个数组进行一次插入排序
示例：
初始数组 8，7，6，5，4，3，2，1
增量为4，分成4个子数组8,4；7,3；6,2；5,1各自插入排序后4,8 ; 3,7 ; 2,6 ; 1,5
此时数组为4,3,2,1,8,7,6,5
增量取2，分为4,2,8,6和3,1,7,5各自排序后2,4,6,8和1,3,5,7
此时2,1,4,3,6,5,8,7
增量为1，进行一次插入排序后得到有序序列1,2,3,4,5,6,7,8

```cpp
void shellSort(vector<int> &arr, int len)
{
    int gap = len / 2;
    while (gap > 0)
    {    //对每个子列表进行插入排序
        for (int i = gap; i < len; i++)
        {
            int temp = arr[i];//当前待插入元素
            int j;
            for (j = i; j >= gap && arr[j - gap] > temp; j -= gap)//在子列表中找到合适的位置插入
            {
                arr[j] = arr[j - gap];
            }
            arr[j] = temp;
        }
        gap /= 2;//减小增量
    }
}
```

时间复杂度取决于增量序列的选择，平均在$$O(n log n)$$到$$O(n^2)$$之间
空间复杂度$$O(n)$$（合并操作需要额外空间来存储）
